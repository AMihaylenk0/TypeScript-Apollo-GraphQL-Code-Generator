### TypeScript Apollo GraphQL Code Generator

## Uses the following technologies:

- TypeScript
- Apollo client
- React
  <br>

## Running Locally:

```
npm install
npm run prestart
npm run start
```

## Folder Structure

```bash
├───public
├───src
│   ├───components
│   └───graphql
│       └───__generated__
└───__generated__
```
